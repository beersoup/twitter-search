
###Getting Started###


```
    In command line
	> git clone https://github.com/beersoup/twitter-search.git
	> cd twitter-search

	Go to https://apps.twitter.com/ and Create New App to get :
	1. Consumer Key
	2. Consumer Secret
	3. Access Token
	4. Access Token Secret

	Create config.js file in the project root
	Copy/Paste these into the config.js and replace with your keys :

	module.exports = {
        consumer_key: 'YOUR CONSUMER KEY HERE',
        consumer_secret: 'YOUR CONSUMER SECRET HERE',
        access_token_key: 'YOUR ACCESS TOKEN HERE',
        access_token_secret: 'YOUR ACCESS TOKEN SECRET HERE'
    }

    In command line
	> npm install
	> node .

	Listening port 8080
```
